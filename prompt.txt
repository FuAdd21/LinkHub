# FIX THE THEME SYSTEM (VERY IMPORTANT)

The theme toggle currently has problems.

When switching themes:

• text colors do not update
• icons stay dark
• backgrounds stay wrong
• grids and cards do not adapt

You must fix this globally.

---

## Implement 3 theme modes

### 1️⃣ Dark Mode (default)

Modern SaaS dark theme.

Example palette:

background:

#0B0C10

cards:

#111218

border:

rgba(255,255,255,0.08)

accent gradient:

purple → pink → blue

---

### 2️⃣ Light Mode

Use a **soft milk background**.

Example:

#F7F6F3

Text must become dark.

Cards must become white.

Borders must adjust.

Icons must switch automatically.

---

### 3️⃣ System Mode

Use:

prefers-color-scheme

Automatically match OS theme.

---

## Technical requirements

Use **CSS variables**:

:root
[data-theme="dark"]
[data-theme="light"]

Every component must use variables.

Examples:

--bg
--card
--text
--border
--accent

Store theme preference in **localStorage**.

When the theme changes, the **entire page must update instantly.**
